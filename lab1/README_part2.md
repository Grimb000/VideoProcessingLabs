# Lab 1 — Part 2: Video Stream Processing

Интерактивная программа для покадровой обработки видео с переключением фильтров.

## Скачать EXE

1. Перейди во вкладку [Actions](../../actions)
2. Открой последний успешный запуск **Build EXE**
3. Внизу страницы найди раздел **Artifacts**
4. Скачай **lab_video_part2_exe** — внутри будет `lab_video_part2.exe`

## Запуск

1. Положи `lab_video_part2.exe` и `fragment.mp4` в **одну папку**
2. Запусти `lab_video_part2.exe`

> ⚠️ Без файла `fragment.mp4` рядом программа не запустится

## Запуск из исходного кода

Установи зависимости:
```bash
pip install opencv-python numpy
```

Запусти:
```bash
python lab_video_part2.py
```

## Управление

| Кнопка | Действие |
|--------|----------|
| Original | Исходный кадр без обработки |
| Gray | Перевод в оттенки серого |
| Blur | Сглаживание гауссовым фильтром |
| HSV | Преобразование в цветовое пространство HSV |
| Pause / Resume | Поставить / снять паузу |

Выход: клавиша **Q**
